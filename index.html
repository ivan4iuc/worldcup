Как играем
Войди под своим именем (имя + личный 4-значный PIN — задаётся при первом входе). Для каждого матча укажи минуту (0–120), на которой ждёшь гол. Менять ставку можно сколько угодно раз до начала матча — после стартового свистка стоп.

До начала матча чужие ставки скрыты 🔒 — каждый видит только свою. После начала — открыты.

Очки начисляются за каждый гол отдельно:
• Точно угадал минуту гола = 3 очка ★
• Ближе всех к этой минуте (но не точно) = 1 очко (при равенстве — каждому из ближайших).
• Если кто-то угадал точно — «ближе всех» в этом раунде не присуждается.

В матче несколько голов → разыгрывается несколько раундов (по 3 очка за гол). Одна твоя ставка может принести очки сразу за несколько голов. Чем больше голов — тем больше очков 😄

• Ставка 0 = прогноз на 0:0. Если матч без голов — поставившим 0 по 3 очка, остальным ноль.

Результаты вносятся автоматически (минуты голов и проход по сетке), админ может поправить вручную. Таблица разбита на вкладки: Групповой этап (сыгран по старым правилам — первый гол), Плей-офф и Общий зачёт.
